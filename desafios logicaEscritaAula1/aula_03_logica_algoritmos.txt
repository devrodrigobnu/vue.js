			Lógica de programação com algoritmos

Diariamente executamos uma série de acõe. Intuitivamente, aquilo que estamos
executando é um algoritmo.
um algoritmo é uma sequência de instruções ordenadas de forma lógica para
a resolução de uma determinada tarefa ou problema.


Algoritmo é uma receita de bolo, de um processo de computador.
Os objetos são manipulados por essas receitas (variaveis)
Um algoritmo é formalmente uma sequência finita de passos que levam 
a execução de tarefa.


	exemplo de algoritmo:
1. escreva o primeiro numero
2. escreva o segundo número
3. Some o primeiro número com o segundo e escreva o resultado


	regras básicas sempre usar o mesmo verbo (imperativo)
exemplo mascar  chiclete
"pegar" chiclete
"retirar" o embrulho
"mascar" o chiclete
"jogar" papel no lixo
