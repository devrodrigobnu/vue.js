PROTOCOLOS DE COMUNICAÇÃO

TCP/IP (Transmission Control Protocol / Internet Protocol)


TCP - > Gerencia como os dados são enviados e recebidos entre dispositivos na rede.

IP -> Lida com o endereçamento e roteamento dos dados, ou seja, como os dados encontram seu caminho através da rede até o servidor.
