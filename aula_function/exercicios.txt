Lista de Exercícios Dupla - Function
1-Crie uma função que imprima "Olá, mundo!" na tela.
2-Crie uma função que receba um número como parâmetro e retorne o seu dobro.
3-Crie uma função que receba dois números como parâmetros e retorne a soma deles.
4-Crie uma função que receba uma string como parâmetro e retorne a quantidade de caracteres dela.
5-Crie uma função que receba uma string como parâmetro e retorne a primeira letra dela.
6-Crie uma função que receba uma string como parâmetro e retorne a última letra dela.
7-Crie uma função que receba uma string como parâmetro e retorne a string invertida.
8-Crie uma função que receba uma string como parâmetro e retorne a string sem espaços em branco.
9-Crie uma função que receba um número como parâmetro e retorne o seu quadrado.
10 - retornar o cubo
11 - retornar quantas vogais tme em uma palavra

Exercícios difíceis:
Crie uma função que receba um número como parâmetro e retorne true se o número for primo e false se não for.
Crie uma função que receba uma string como parâmetro e retorne true se a string for um palíndromo e false se não for.
