Essas são apenas algumas das pseudoclasses mais comuns em CSS. Eles são usados ​​para aplicar estilos específicos a elementos com base em diferentes estados ou critérios de seleção, tornando a estilização de páginas da web mais dinâmica e interativa

1):hover = Aplicado quando o mouse está sobre o elemento'botão'. Usado para criar efeitos como mudar a cor de um link quando o mouse passa sobre ele ou um botão

Passe o mouse aqui

2) :active = Aplicado quando o elemento está sendo clicado ou interagido de alguma forma. Muitas vezes usado para destacar botões ou links durante a interação do usuário.



3):focus = Aplicado quando um elemento recebe um elemento foco, geralmente usado em campos de entrada de formulário para indicar que o campo está ativo e pronto para entrada



4):nth-child(odd) = Permite selecionar elementos com base em sua posição dentro de um contêiner. Por exemplo, :nth-child(n)selecione todos os elementos impossíveis dentro de um contêiner
