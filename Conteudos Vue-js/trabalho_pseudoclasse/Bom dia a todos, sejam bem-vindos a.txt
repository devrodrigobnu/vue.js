1- Agora vamos dar uma olhada na estrutura basica 
de HTML e CSS usando pseudo-elemento placeholder.
Aqui eu criei um formulário simples onde nós temos um input para o usuario com id "name"
o placeholder inicialmente é um texto temporário exibido dentro de um campo de input.
Basicamente ele fornece uma instrução, que nesse caso é "insira seu nome".
Quando o usuário começa a digitar, o texto do placeholder desaparece automaticamente.
